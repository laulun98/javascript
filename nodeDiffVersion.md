# NODE DIFERENTES VERSIONES PROYECTOS

Cuando no estamos en nuestro ordenador y necesitamos otra versión de NODE, tenemos dos opciones.
1. NVM (NOde Version Manager): https://github.com/nvm-sh/nvm#installing-and-updating
2. VOLTA: https://volta.sh/

## USANDO VOLTA

- instalacion: ''' volta install node@22 '''
- node version: ''' volta pin node@<node_version>'''

## PARTICULARIDADES

Te permite instalar paquetes (o librerías (idk)) y que estaás estén pineadas a ciertas versiones de node, que no cambia a no ser que lo indiques.
